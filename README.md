# Divulgación Científica - Aplicación de subida de documentos

Pequeña aplicación para subir y mostrar documentos científicos (PDF, imágenes, etc.). Pensada como punto de partida para una página de divulgación científica.

Requisitos
- Node.js 14+ instalado

Instalación

```bash
cd /ruta/al/proyecto/Anteproyecto
npm install
npm start
```

Uso
- Abre http://localhost:3000
- Sube archivos usando el formulario.
- La lista de documentos aparece en la página principal; puedes ver los PDFs embebidos o descargar el archivo.

Notas y extensiones recomendadas
- Validar tipos MIME y tamaño máximo en el backend para evitar subir archivos no deseados.
- Añadir autenticación si solo ciertos usuarios deben subir.
- Añadir una base de datos para almacenar metadatos y búsquedas.
